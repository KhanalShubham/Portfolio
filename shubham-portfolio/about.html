About Shubham

I’m Shubham Khanal, a full-stack developer with a strong focus on UI/UX design and data science. I enjoy building systems that look clean, feel smooth, and are backed by data-driven logic.

I’m currently a Bachelor student (6th semester), actively working on academic, research, and side projects — from computer-vision-based tennis analytics to remote-controlled cars and scholarship platforms for students.

Core Skills & Tools

Full-stack (MERN) Node.js JavaScript / TypeScript Python R & pandas Data Science & Analytics Computer Vision (YOLOv8) UI/UX Design Figma Canva Git & GitHub

Highlighted Projects

Tennis Vision – Shot Detection & Analytics

Python · YOLOv8 · Computer Vision · Data Pipeline

Ran object detection on tennis match footage using the Ultralytics YOLOv8 model to detect the player, racket, and ball in every frame. The system stores class labels, confidence scores, and bounding boxes as structured data, forming the base for shot tracking, performance analysis, and a future end-to-end tennis analytics pipeline.

Remote RC Car Simulator – TECH X ELEVATE

Raspberry Pi 4 · Logitech G29 · PWM · Robotics

Designed a realistic RC car controlled via a Logitech G29 steering wheel and pedals, powered by a Raspberry Pi 4. Features precise acceleration, braking, steering, and gear control for an immersive driving experience. The project blends entertainment and education, demonstrating robotics, programming, and control systems.

ReFoodify – Reducing Food Waste

Figma · UI/UX Concept

Concept app designed in Figma to reduce food wastage by reusing and redistributing surplus food. Focused on clear user journeys for donors, NGOs, and receivers, with emphasis on social impact and accessibility.

Khana Baanam – Recipe Teacher App

Android · Kotlin

Android app with two modes: a user side to explore recipes and an admin side to upload and manage recipes and ingredients. Built to make learning cooking easier and more organized.

Online Auction System

Java · Apache NetBeans · Group Project

Developed a multi-user auction platform supporting user registration, item listing, live bidding with timers, automatic winner assignment, sold-items history, client lists, and analytical statistics of costs and earnings for both bidders and auctioneers.

ApplyBro – Scholarship Helper

Web · Student-centric Platform

A website concept to help students discover the best scholarships for them, bringing application information, eligibility criteria, and guidance into one simple interface.
